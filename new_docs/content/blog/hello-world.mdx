Welcome to the DeepEval blog. This is a placeholder post used to wire up the blog scaffold — delete or replace it when real content lands.

## Why this post exists

It demonstrates the full blog frontmatter:

- `title` and `description` — same as any docs page.
- `date` — required on real posts; drives the byline shown above.
- `authors` — an array of IDs registered in `lib/authors.ts`. A typo here fails the build with a Zod error so you never ship a broken byline.

## Same prose, same components

Blog posts have access to every globally-registered component, so you can use callouts, image displayers, steps, and everything else without importing anything:

:::tip[Good to know]
This callout, the image frame, the typography, and the copy-markdown button all behave identically to docs pages.
:::
